# Prompt 03 — Caption Generator

> **วัตถุประสงค์:** สร้าง Caption สำหรับทุก Platform จาก Script ที่อนุมัติแล้ว
>
> **วิธีใช้:** Copy Prompt → วางใน ChatGPT หรือ Claude → กรอก Input → ตรวจ Output → ใช้ใน Publishing Calendar

---

## Input ที่ต้องกรอกก่อนใช้งาน

| Input | คำอธิบาย | ตัวอย่าง |
|-------|---------|---------|
| **Script** | Script หรือสรุปเนื้อหาหลัก | `วิดีโอสอนการใช้ Sauna สำหรับมือใหม่ 5 tips` |
| **Platform** | เลือก Platform ที่ต้องการ | `TikTok, Facebook, Instagram` |
| **Brand** | ชื่อแบรนด์ + Tone | `Harmony Sauna — อบอุ่น ผ่อนคลาย` |
| **CTA** | สิ่งที่ต้องการให้ผู้ชมทำ | `เพิ่ม LINE @HarmonySauna เพื่อจอง` |
| **Language** | ภาษาที่ต้องการ | `ไทย` / `อังกฤษ` / `ไทย-อังกฤษ` |
| **Keywords** | คีย์เวิร์ดสำคัญ | `sauna, พัทยา, wellness, ผ่อนคลาย` |

---

## Prompt Template (Copy-Paste Ready)

```
คุณเป็น Social Media Copywriter ผู้เชี่ยวชาญด้านการเขียน Caption สำหรับหลาย Platform

ฉันต้องการ Caption สำหรับ:
- เนื้อหาวิดีโอ/โพสต์: [SCRIPT_SUMMARY]
- Brand: [BRAND_NAME] — Tone: [TONE]
- CTA หลัก: [CTA]
- ภาษา: [LANGUAGE]
- Keywords ที่ต้องมี: [KEYWORDS]

กรุณาสร้าง Caption สำหรับแต่ละ Platform ดังนี้:

---

**FACEBOOK CAPTION:**
(ความยาว 100-300 คำ, เริ่มด้วยประโยคที่ดึงดูด, ใส่ emoji พอประมาณ, จบด้วย CTA ชัดเจน)

---

**INSTAGRAM CAPTION:**
(ความยาว 150-300 คำ, Hook ใน 2 บรรทัดแรก, ใช้ line breaks, จบด้วย CTA + ขึ้นบรรทัดใหม่ก่อน hashtags)

---

**TIKTOK CAPTION:**
(ความยาวไม่เกิน 150 คำ, สั้นกระชับ, ใส่ emoji, จบด้วย CTA + 3-5 hashtags หลักในคำบรรยาย)

---

**YOUTUBE SHORTS TITLE:**
(ความยาวไม่เกิน 60 ตัวอักษร, ใส่ Keyword หลัก, ดึงดูดคลิก)

**YOUTUBE SHORTS DESCRIPTION:**
(ความยาว 100-200 คำ, ใส่ keyword, timestamp ถ้ามี, CTA, ลิงก์)

---

**X (TWITTER) POST:**
(ความยาวไม่เกิน 280 ตัวอักษร, กระชับ, มี punch, CTA + 2-3 hashtags)

---

**HASHTAGS (15-20 อัน):**
แบ่งเป็น 3 กลุ่ม:
- กลุ่ม Brand: [5 hashtags]
- กลุ่ม Niche: [5-7 hashtags]
- กลุ่ม Popular: [5-8 hashtags]

---

**CTA VARIANTS (3 แบบ):**
1. Soft CTA: (ไม่กดดัน เชิญชวน)
2. Direct CTA: (ตรงประเด็น ชัดเจน)
3. Value CTA: (มอบสิ่งตอบแทน เช่น ฟรี/พิเศษ)
```

## Platform Caption Length Guide

| Platform | ความยาวแนะนำ | หมายเหตุ |
|----------|------------|---------|
| Facebook | 100-300 คำ | ใส่ emoji ช่วยแบ่ง section |
| Instagram | 150-300 คำ | 2 บรรทัดแรกต้องดึงดูด |
| TikTok | ไม่เกิน 150 คำ | สั้น กระชับ ตรงประเด็น |
| YouTube Shorts Title | ไม่เกิน 60 ตัวอักษร | ใส่ keyword |
| YouTube Shorts Description | 100-200 คำ | ใส่ timestamp |
| X (Twitter) | ไม่เกิน 280 ตัวอักษร | กระชับที่สุด |

## Checklist ก่อนใช้ Caption

- [ ] ตรวจชื่อแบรนด์ถูกต้อง

- [ ] Hashtags เกี่ยวข้องกับคอนเทนต์

- [ ] ความยาวเหมาะกับ Platform
