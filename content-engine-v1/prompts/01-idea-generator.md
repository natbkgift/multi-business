# Prompt 01 — Idea Generator

> **วัตถุประสงค์:** สร้าง Content Ideas 10 ชิ้นสำหรับแบรนด์และ Platform ที่ต้องการ
>
> **วิธีใช้:** Copy Prompt Template ด้านล่าง → วางใน ChatGPT หรือ Claude → กรอก Input → Copy Output ลง Idea Bank

---

## Input ที่ต้องกรอกก่อนใช้งาน

| Input | คำอธิบาย | ตัวอย่าง |
|-------|---------|---------|
| **Brand** | ชื่อแบรนด์และคำอธิบายสั้นๆ | `Harmony Sauna — สปาและซาวน่าในพัทยา` |
| **Topic** | หัวข้อหลักที่อยากสร้างคอนเทนต์ | `ประโยชน์ของการใช้ Sauna` |
| **Audience** | กลุ่มเป้าหมายเฉพาะ | `ผู้หญิงไทยอายุ 25-40 ที่อยู่ในพัทยา` |
| **Platform** | TikTok / Facebook / YouTube / Instagram | `TikTok` |
| **Goal** | เพิ่ม awareness / leads / sales / engagement | `เพิ่ม awareness + leads (จอง LINE)` |
| **Tone** | บรรยากาศที่ต้องการ | `อบอุ่น ผ่อนคลาย ชวนลอง` |
| **Avoid** | สิ่งที่ห้ามพูดถึง | `ห้ามอ้างสรรพคุณทางการแพทย์` |

---

## Prompt Template (Copy-Paste Ready)

```
คุณเป็น Content Strategist ผู้เชี่ยวชาญด้าน Short-form content สำหรับ Social Media ในประเทศไทย

ฉันต้องการ Content Ideas จำนวน 10 ชิ้น สำหรับ:
- Brand: [BRAND_NAME] — [BRAND_DESCRIPTION]
- Topic: [TOPIC]
- Target Audience: [AUDIENCE]
- Platform: [PLATFORM]
- Goal: [GOAL]
- Tone: [TONE]
- สิ่งที่ห้ามพูดถึง: [AVOID]

สำหรับแต่ละ Idea ให้ระบุในรูปแบบ:

**Idea [ลำดับ]:**
- Hook (0-3 วินาที): [ประโยคเปิดที่ดึงดูดทันที]
- Angle Type: [Education / Entertainment / Testimonial / Problem-Solution / Behind the Scene / Promotion / Trend / Comparison / How-to / Storytelling]
- CTA: [สิ่งที่ต้องการให้ผู้ชมทำ]
- Suggested Format: [Short-form 30s / Short-form 60s / Short-form 90s / Reel / Static Post / Carousel / Story / Long-form]
- Risk Note: [ข้อควรระวังด้าน compliance หรือ legal]
- Score (1-10): [คะแนนประเมินศักยภาพ พร้อมเหตุผลสั้นๆ]

กรุณาสร้าง Idea ที่หลากหลาย Angle Type ไม่ซ้ำกัน และเน้น Hook ที่หยุดนิ้วผู้ชมได้ทันที
```

## ตัวอย่างการใช้งาน — Harmony Sauna

**Input ที่กรอก:**
- Brand: Harmony Sauna — ซาวน่าและสปาระดับพรีเมียมในพัทยา
- Topic: ประโยชน์และประสบการณ์ Sauna สำหรับมือใหม่
- Audience: ผู้หญิงไทยอายุ 25-40 ที่ไม่เคยลอง Sauna
- Platform: TikTok
- Goal: เพิ่ม awareness + นำไปสู่การจอง LINE
- Tone: อบอุ่น ผ่อนคลาย ชวนลอง ไม่กดดัน
- Avoid: ห้ามอ้างสรรพคุณการรักษาโรคใดๆ

**ตัวอย่าง Output:**

> **Idea 1:**
> - Hook: "ครั้งแรกที่เข้า Sauna ฉันคิดว่าจะทนไม่ได้... จนนาทีที่ 3"
> - Angle Type: Storytelling
> - CTA: เพิ่ม LINE เพื่อจอง Trial Session ราคาพิเศษ
> - Suggested Format: Short-form 60s
> - Risk Note: ห้ามอ้างว่า Sauna "รักษา" อะไร
> - Score: 9/10 — Hook แบบ personal story ดี มากบน TikTok

> **Idea 2:**
> - Hook: "5 สิ่งที่คนไม่รู้เกี่ยวกับ Sauna ก่อนเข้าครั้งแรก"
> - Angle Type: Education
> - CTA: Save วิดีโอนี้ไว้อ่านก่อนมา
> - Suggested Format: Short-form 60s
> - Risk Note: ตรวจ Fact ก่อน publish
> - Score: 8/10 — List format ทำได้ง่าย share ได้ดี

## ตัวอย่างการใช้งาน — AMP Pattaya

**Input ที่กรอก:**
- Brand: AMP Pattaya — ที่ปรึกษาด้านอสังหาริมทรัพย์พัทยาสำหรับชาวต่างชาติ

- Audience: ชาวต่างชาติที่อาศัยอยู่ในพัทยาหรือสนใจลงทุน

- Tone: Professional แต่เข้าถึงได้ ไม่ formal เกินไป
- Avoid: ห้ามรับประกัน ROI หรือผลตอบแทนแน่นอน

**ตัวอย่าง Output:**

> **Idea 1:**
> - Hook: "Can foreigners really own property in Thailand? The answer surprised me."
> - Angle Type: Problem-Solution
> - CTA: Comment "GUIDE" to get our free foreign buyer checklist
> - Suggested Format: Short-form 90s
> - Risk Note: ระบุชัดว่าเป็นข้อมูลทั่วไป ไม่ใช่คำแนะนำทางกฎหมาย
> - Score: 9/10 — Common question ที่ target audience มักถาม

> - CTA: Save แล้ว share ให้เพื่อนที่ยังทำงานหนักเกินไป
> - Suggested Format: Short-form 60s

## Checklist ก่อนนำ Idea ลง Idea Bank

- [ ] ทุก Idea มี Hook ชัดเจน
- [ ] Angle Type ไม่ซ้ำกันเกิน 3 ชิ้นใน 10 ชิ้น
- [ ] Risk Note กรอกทุก Idea
- [ ] Score ≥ 6 ก่อน Selected
- [ ] ตรวจกับ Brand Playbook ว่าไม่ขัด Do/Don't
