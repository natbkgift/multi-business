# Prompt 02 — Script Writer

> **วัตถุประสงค์:** เขียน Script คอนเทนต์แบบครบถ้วนจาก Approved Idea
>
> **วิธีใช้:** Copy Prompt Template → วางใน ChatGPT หรือ Claude → กรอก Input → Review Output → ส่ง Script Reviewer

---

## Input ที่ต้องกรอกก่อนใช้งาน

| Input | คำอธิบาย | ตัวอย่าง |
|-------|---------|---------|
| **Approved Idea** | Hook + Angle + CTA จาก Idea Bank | `Hook: "ครั้งแรกที่เข้า Sauna..."` |
| **Brand** | ชื่อแบรนด์ + Tone of Voice | `Harmony Sauna — อบอุ่น ผ่อนคลาย` |
| **Platform** | Platform เป้าหมาย | `TikTok` |
| **Length** | ความยาวที่ต้องการ | `60 วินาที` |
| **Tone** | บรรยากาศ Script | `สบายๆ เหมือนเล่าให้เพื่อนฟัง` |
| **Key Points** | 3-5 ประเด็นหลักที่ต้องใส่ | `1. ความรู้สึกครั้งแรก 2. เทคนิคการหายใจ 3. CTA จอง` |
| **Avoid** | สิ่งที่ห้ามพูด | `ห้ามอ้างสรรพคุณทางการแพทย์` |

---

กรุณาเขียน Script ตามโครงสร้าง:

**[HOOK — 0-3 วินาที]**

**[CTA — 5 วินาทีสุดท้าย]**
(สิ่งที่ต้องการให้ผู้ชมทำ ชัดเจน ไม่เกิน 1 CTA)

**[B-ROLL SUGGESTIONS]**

- Shot 2:
- Shot 3:

- ข้อความที่ควร overlay ตอนต้น:
- ข้อความที่ควร overlay ตอนกลาง:
- ข้อความ CTA:

**[COMPLIANCE CHECKLIST]**

- [ ] ไม่มีตัวเลขรับประกัน ROI
- [ ] ราคา (ถ้ามี) เป็นปัจจุบัน

## ตัวอย่าง Script จริง — AMP Pattaya (90 วินาที Facebook)

**Input:**
- Hook: "Can foreigners really own property in Thailand? The answer surprised me."
- Brand: AMP Pattaya — Professional, approachable
- Platform: Facebook / YouTube | Length: 90s
- Key Points: กฎหมายคนต่างชาติ, Freehold vs Leasehold, ขั้นตอนการซื้อ
- Avoid: ห้ามรับประกัน ROI หรือราคาที่แน่นอน

**Script Structure:**

**[HOOK — 0-3s]:** "Most foreigners think they can't buy property in Thailand. They're wrong."
**[SETUP — 3-10s]:** "In this video I'll show you exactly what foreigners CAN legally own — and what the process looks like."
**[BODY — 10-75s]:**
- Point 1 (Freehold Condo): Foreigners can own condos freehold — up to 49% of building
- Point 2 (Leasehold): Land via 30+30+30 year lease
- Point 3 (Process): Reserve → Due Diligence → Transfer
- Pattern Interrupt: "But here's what most agents won't tell you..."
**[CTA — 75-90s]:** "Comment 'GUIDE' and I'll send you our free Foreign Buyer Checklist"
